# AI Penpal Letterbox 测试文档

本文档用于组员测试本项目。项目技术栈为 Next.js + Supabase，主要功能包括：用户注册登录、浏览/邂逅笔友、写信寄出、等待回信、查看收件箱、查看笔友详情和个人页。

## 0. 测试前准备

### 0.1 环境准备

| 项目 | 说明 |
| --- | --- |
| Node.js | 建议使用项目当前可正常运行的版本 |
| 依赖安装 | 执行 `npm install` |
| 环境变量 | 确认 `.env.local` 中 Supabase 相关变量已配置 |
| 启动项目 | 执行 `npm run dev` |
| 访问地址 | 浏览器打开 `http://localhost:3000` |

### 0.2 测试账号准备

建议准备 2 个 Supabase 测试账号：

| 账号 | 用途 |
| --- | --- |
| 账号 A | 正常注册、登录、写信、收信 |
| 账号 B | 验证不同用户之间的数据隔离 |

测试时不要使用真实用户隐私信息。

### 0.3 通用通过标准

每个页面都需要同时检查：

- 页面可以正常打开，没有白屏。
- 控制台没有明显报错。
- 按钮、输入框、链接可以正常点击或输入。
- 页面文字、图片、卡片布局没有明显重叠或溢出。
- 刷新页面后核心数据仍然存在。
- 手机尺寸下页面仍然可读、可操作。
